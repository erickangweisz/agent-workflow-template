# Uso Con Copilot

Copilot no tiene el mismo concepto nativo de skill que Codex.

## Forma recomendada de uso

1. Abrir `agents/bootstrap.md`
2. Iniciar una sesion nueva en Copilot
3. Pegar el bootstrap una vez al comenzar la conversacion
4. Mantener el flujo Arquitecto -> Frontend -> QA dentro de la misma sesion

## Recomendacion

Usar esta plantilla como base de prompts y handoffs, no como mecanismo automatico del cliente.
